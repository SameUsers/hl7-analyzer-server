# HL7 Analyzer Server

> **Асинхронный Python-сервер для приема HL7 v2.x сообщений от ветеринарных анализаторов.**
>
> Принимает HL7 по TCP/MLLP, парсит результаты, нормализует данные и экспортирует в JSON для интеграции с ветеринарными и лабораторными информационными системами.

---

## Поддерживаемые анализаторы

- ✅ Vet5160 (гематология)
- ✅ Accent M320 (биохимия)
- ✅ Seamaty SMT-120VP (биохимия)

---

## Возможности

- Асинхронный TCP-сервер (AsyncIO)
- HL7 v2.x парсер с MLLP-обрамлением
- Нормализация данных под каждый анализатор
- Валидация через Pydantic v2
- Экспорт в JSON
- Расширяемая архитектура
- Dependency Injection
- Python 3.12+

## Поток данных

1. Анализатор отправляет HL7-сообщение по TCP
2. Сервер получает байтовый поток
3. Детектор выделяет сообщение по MLLP-маркерам (`0x0B ... 0x1C0D`)
4. Парсер преобразует строку в структурированные сегменты
5. Билдер нормализует данные под конкретный анализатор
6. Pydantic валидирует результат
7. Создается типизированный DTO (`AnalyzeResult[T]`)
8. Хранилище сохраняет результат
